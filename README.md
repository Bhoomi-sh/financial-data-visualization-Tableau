# Financial Data Visual Analytics (Tableau)

## Overview
This project focuses on visual data analysis and exploratory data mining using Tableau Desktop.
The objective is to analyze financial performance data and uncover patterns through interactive visualizations.

## Key Work
- Created multiple Tableau worksheets using different chart types
- Built interactive dashboards and stories
- Used Tableau Extracts (.hyper) for performance
- Performed co-occurrence analysis to identify relationships between variables
- Linked findings to association rule mining concepts

## Tools & Technologies
- Tableau Desktop
- Data Visualization & Dashboards
- Visual Analytics
- Co-occurrence Analysis
